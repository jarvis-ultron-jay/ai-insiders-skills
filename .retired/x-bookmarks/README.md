# X Bookmarks — Stop Hoarding, Start Acting

Turn your X/Twitter bookmarks into actionable work. Stop letting saved tweets pile up — process, categorize, and extract value from your bookmark collection.

## The Problem

You bookmark tweets with good intentions but never revisit them. They become digital clutter. Great insights get buried. Actionable advice goes unacted upon. Your bookmarks become a graveyard of missed opportunities.

## The Solution

This skill transforms passive bookmark hoarding into active knowledge building:

- **Process systematically** — Regular review and smart categorization
- **Extract actionable insights** — Turn tweets into tasks, ideas, and content
- **Create digestible summaries** — Weekly digests instead of overwhelming backlogs
- **Build knowledge connections** — Link related tweets and concepts  
- **Enable smart retrieval** — Find bookmarks when you actually need them

## Features

- **Automated Processing** — Bulk categorization and analysis
- **Smart Summarization** — Weekly/monthly digests of top bookmarks
- **Action Extraction** — Convert bookmarks into concrete next steps
- **Pattern Recognition** — Identify trends in what you save
- **Content Generation** — Turn bookmark themes into content ideas
- **Knowledge Building** — Create personal wiki from bookmark insights
- **Search & Retrieval** — Find relevant bookmarks by topic or project

## Install

```bash
npx clawhub install x-bookmarks
```

## Setup

Requires authentication via `bird` CLI:

```bash
# Install bird CLI
npm install -g bird-cli

# Authenticate with X/Twitter
bird auth

# Test access  
bird bookmarks list --limit 5
```

## Workflow Patterns

### Daily Quick Review (5 minutes)
- Scan last 24 hours of bookmarks
- Flag anything urgent or actionable
- Quick high/medium/low value rating

### Weekly Deep Dive (30 minutes)  
- Categorize all bookmarks by topic
- Extract key insights and action items
- Create weekly digest for review
- Archive or delete low-value saves

### Monthly Knowledge Building (60 minutes)
- Analyze bookmark patterns and trends
- Build topic-based knowledge summaries
- Update personal knowledge base
- Generate content calendar from themes

### Project-Based Retrieval (As needed)
- Search bookmarks by keyword/topic
- Create project-specific resource lists
- Extract relevant quotes and insights
- Build research bibliography
